The Chefs: Digital Cookbook Project

Project Overview

Digital Cookbook is an integrated recipe management system designed for home chefs who discover new cooking ideas on short-form video platforms, especially TikTok.
The platform streamlines the process from recipe discovery to cooking and shopping by automatically extracting recipe details, saving and organizing them, and creating unified grocery lists.
Our goal is to eliminate the hassle of manual note-taking, fragmented organization, and inefficient grocery shopping by providing a seamless, automated experience.

Key Features

  • Auto-extract recipe: Paste a TikTok link to automatically extract recipe title, ingredients, and instructions using AI.
  • Edit recipes: Users can review and customize auto-generated recipes for accuracy and personalization.
  • Organized cookbook: Recipes are saved in a searchable, categorized library with filtering options.
  • Grocery list integration: Instantly create a unified grocery list from selected recipes, with duplicate ingredients merged.
  • Secure account management: Register, log in/out, and manage personal recipes and lists safely.
  • Mobile-friendly: All features are optimized for use in the kitchen on modern browsers and mobile devices.
  • Original video linking: Each recipe card preserves the original TikTok video for easy reference and creator attribution.

Database Overview

  • Database Type: PostgreSQL (running in Docker containers)
  • Main tables: users, recipes, ingredients, categories, grocery_lists, grocery_list_items
Excerpt from docker-compose.yml:
services:
  db:
    image: postgres:15
    container_name: digitalcookbookdb
    environment:
      POSTGRES_USER: chefuser
      POSTGRES_PASSWORD: cookbook123
      POSTGRES_DB: digitalcookbook
    ports:
      - "5432:5432"
    volumes:
      - db_data:/var/lib/postgresql/data
      - ./init-db:/docker-entrypoint-initdb.d
